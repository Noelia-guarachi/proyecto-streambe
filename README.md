# proyecto-streambe
Proyecto web creado solo con HTML y CSS que simula un museo virtual con distintas salas temáticas de arte. El objetivo es ofrecer una experiencia inmersiva y visualmente atractiva, demostrando el uso de estructuras semánticas, diseño responsive y efectos modernos.
avance noelia guarachi: Creé la estructura base del Museo Virtual Interactivo usando HTML5.
Incluí la portada de bienvenida, tres salas temáticas (pinturas, esculturas y arte moderno).
En esta etapa enfoqué el código en la organización y en que el contenido esté bien estructurado antes de aplicar estilos.
